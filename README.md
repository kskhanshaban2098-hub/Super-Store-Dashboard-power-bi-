# 📊 Super Store Power BI Dashboard

## 📌 Overview

This Power BI dashboard provides an interactive analysis of Super Store sales performance. It helps users monitor sales, profit, revenue, customer trends, and regional performance to support data-driven business decisions.

## ✨ Key Features

- 📈 Sales Analysis
- 💰 Profit Analysis
- 📊 Revenue Insights
- 🌍 Regional Performance
- 🛍 Category-wise Sales
- 📦 Order Analysis

## 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel

## 🚀 Purpose

This project demonstrates how Power BI can be used to build interactive dashboards for business intelligence and sales performance analysis.
